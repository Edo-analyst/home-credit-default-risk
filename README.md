# Home Credit Default Risk — Binary Classification

## Project Overview
End-to-end machine learning project to predict loan default probability using real-world data from Home Credit Group.

## Business Problem
Home Credit wants to predict which loan applicants are likely to default, enabling better lending decisions and reduced financial risk.

## Tech Stack
- Python, pandas, numpy
- scikit-learn (Pipeline, ColumnTransformer)
- XGBoost, LightGBM
- SHAP for model explainability
- matplotlib, seaborn

## Project Structure
- `data/` — raw and processed datasets
- `notebooks/` — EDA, preprocessing, modeling, explainability
- `src/` — reusable Python modules
- `models/` — saved trained models
- `reports/figures/` — saved plots
